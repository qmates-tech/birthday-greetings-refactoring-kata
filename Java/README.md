# Birthday Greetings Kata - Java Version

Java version of the Birthday Greetings Kata.

## What's this?

This is a simple refactoring exercise that is meant to teach something about refactoring, dependency inversion and dependency injection.

The original documentation is at root level: see the [README file](../README.md) and the [doc directory](../doc/).

## Setup

1. Make sure you have Java JDK (at least version 17) installed
2. Compile and run the tests with:

```
./mvnw clean test
```

## How to start and continue

Follow the root level [README file](../README.md).
